🔗 BLOCKCHAIN BASICS
⬡ BLOCKCHAIN
Distributed ledger of sealed blocks. Once written, data cannot be changed — ideal for proof of work.
🎨 NFT (ERC-721)
Non-Fungible Token — a unique digital asset. Each invoice = one NFT with its own permanent ID.
📜 SMART CONTRACT
Self-executing code on blockchain — handles minting, payments, registry. No middlemen needed.
🌐 IPFS
Decentralized storage. Invoice metadata pinned via Pinata — no single server can delete it.
💳 WALLET
Your identity on blockchain. MetaMask signs transactions — like a cryptographic digital signature.
🔺 POLYGON
Layer-2 Ethereum chain — fast and cheap. Amoy Testnet for development. Tiny gas fees.
✨ CORE FEATURES
🤖 AI Invoice Generation
Plain English → AI extracts all details. Conversational editing. Rate estimation included.
🎨 NFT Minting
Invoice → ERC-721 NFT on Polygon. NFT ID = Invoice #. Stored on IPFS permanently.
💸 Crypto Payments
USDT stablecoin. Direct wallet transfer. Zero fees. Real-time on-chain verification.
📊 Dashboard
NFT Gallery. Payment status checker. Token whitelist. User registry lookup.
🔐 On-Chain Identity
Portfolio URL on blockchain. Verifiable freelancer profile. Anyone can verify.
